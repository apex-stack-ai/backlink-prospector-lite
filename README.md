# Backlink Prospector Lite — Claude Skill

Free link building assistant for Claude. Find guest post opportunities and draft personalized outreach pitches.

## What It Does

Give Claude your website URL and niche, and it will:

1. **Search for guest post targets** using proven search query templates
2. **Qualify prospects** — skip link farms and low-quality sites
3. **Present up to 5 targets** with DA estimates, topics, submission methods
4. **Draft one personalized outreach email** for your top target

## Install

1. Download `SKILL.md` from this repo
2. Place it in your Claude skills directory
3. Ask Claude: "Find guest post opportunities for my [niche] site at [URL]"
